
# Geography Quiz (Python)
Mini juego de preguntas por consola desarrollado en Python.
El jugador debe responder correctamente para avanzar mientras gestiona un sistema de vidas.


## ¿Como usar el juego?

Requisitos:
- Python 3

Ejecución:
1. Buscar simbolo del sistema / cmd
2. Buscar la carpeta donde se haya guardado el archivo
3. Ejecutar el programa con python3
4. python3 geography_quiz.py

## Tecnologías usadas

-Python 3
- Librería random

## Habilidades aprendidas

- Uso de bucles (while)
- Validacion de entradas del usuario
- Control de flujo con condiciones
- Logica de juego por consola
